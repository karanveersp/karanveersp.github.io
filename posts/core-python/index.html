An Introduction to Python

Python is a general purpose, inerpreted, object-oriented and structural scripting language. For me, it’s the swiss army knife of programming languages.

Like any programming enthusiast, I’m always excited to learn a new language. Over the past year, I’ve learned the basics of Haskell, F#, Groovy, and Go. Each one of these is amazing and fun to use. But the fact remains that the one language with which I’ve logged the most hours till date is Python.

Python was the 4th programming language I was exposed to after C++, C# and Java. Coming from these C-style languages to Python was, believe it or not, somewhat intimidating but I loved it as soon as I learned it.

Programming in Python feels like flying. It feels fast. Stream of consciousness fast. As soon as a logical step assembles in your thought, you can manifest it in Python with ease. It’s something truly special and will boost your productivity as a programmer by orders of magnitude.

Although it’s a fact that programs run slower than those in C-style languages, I never physically noticed it after over a year of using it. The developer experience with Python is incredibly responsive. Python isn’t designed for low level programming where speed and memory are primary concerns. But it’s more than capable of handling anything else, from web servers, to command line utilities, to desktop applications.
